# gifos
creación de una aplicación web interactiva para compartir GIFs animados, poniendo en práctica los fundamentos de la programación y el uso de JavaScript mediante la obtención de datos de la API de Giphy.com, trabajando con asincronismo y aprovechando herramientas nativas del explorador como la obtención de imágenes a través de la webcam.
En esta decidi practicar css puro y dejar a un lado SASS
